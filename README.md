## What's it about? 
This is a basic book recommendation app to get me working with API's and awkward colour palettes.

- Open Library API is used to search book title and author
- Favicon sourced from  <a target="_blank" href="https://icons8.com/icon/30066/book-shelf">Book Shelf</a> icon by <a target="_blank" href="https://icons8.com">Icons8</a>
- <a target="_blank" href="https://github.com/VincentGarreau/particles.js">Vincent Garreau particles</a> used for background
- <a target="_blank" href="https://github.com/w3slley/bookcover-api">w3slley API</a> used for searching book covers
